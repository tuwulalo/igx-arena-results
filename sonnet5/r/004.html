NEXUS OS — справка и деплой

Симулятор игровой консоли на Three.js + GSAP + Web Audio API. Один self-contained HTML файл, 60 FPS, единый requestAnimationFrame-цикл.

Управление

Меню: ← → выбор карточки, Enter/клик — запуск, Esc — выход в меню.
Minecraft Mini: WASD — ходьба/поворот, ЛКМ — сломать блок перед собой, ПКМ — поставить блок.
GTA Chase Mini: ← → — смена полосы, уворачивайтесь от встречных машин.
UFC Box Mini: Space — удар, W/S — приблизиться/отступить, следите за HP.

Запуск локально

1. Сохраните файл как index.html
2. Откройте его в любом современном браузере
   (двойной клик или: npx serve .)
Никаких зависимостей и сборки не требуется —
все библиотеки подключаются через CDN.

Деплой в один клик

Vercel:
  vercel deploy --prod   (в папке с index.html, через Vercel CLI)
  или перетащите файл на vercel.com/new

Netlify:
  Перетащите index.html на app.netlify.com/drop

GitHub Pages:
  1. Создайте репозиторий, добавьте index.html в корень
  2. Settings → Pages → Source: branch main / root
  3. Сайт будет доступен по адресу
     https://<username>.github.io/<repo>/
