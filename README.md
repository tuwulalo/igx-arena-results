# IGX Тест-Арена : результаты финального теста

Статическая галерея результатов: 109 генераций, 11 заданий, 10 моделей.
Каждая карточка - самодостаточный HTML, сгенерированный моделью по заданию, с метриками: время генерации, токенов/с, цена, через что сделано (раннер).

- `index.html` - галерея с ленивой загрузкой (видимые превью рендерятся, ушедшие из вида выгружаются).
- `r/NNN.html` - сырые результаты моделей (открываются по клику в карточке).

Токенов/с - реальный замер. Время и цена - оценка (выходные токены = размер HTML / 4; тарифы OpenRouter, снимок 2026-06).
